草地编辑器-草数据的读写

2023-03-27
草地编辑器
8.6k words
  • GrassEditor

Desc:

草地编辑器之缝合怪

草地数据的存储：

通过将每个草的位置，存储到Mesh中，用Mesh中的顶点去记录每个草的位置、法线、uv、顶点色信息

效果


需要存储的数据

  1. 顶点位置信息
  2. 顶点色
  3. 法线信息
  4. 草的宽高信息
对 GrassData 的操作

随机草的高度和宽度，使草看起来更自然

在设置草数据的时候，来随机草的宽高

在绘制草的时候，来随机草的宽高，在 compute shader 中改变


网格的操作

Mesh Create
Data Init
从Mesh中获取数据初始化类，用于后续的编辑操作

编辑器的操作

对于编辑器，定义了几种状态

  1. 添加草状态 GrassPainterAddState
  2. 编辑草状态 GrassPainterEditState
  3. 删除草状态 GrassPainterRemoveState
定义接口

定义状态需要实现

  1. 当鼠标按下时的处理
  2. 当鼠标拖拽时的处理
IGrassState
使用
添加草的状态

步骤：

  1. 计算草的位置，通过射线检测，得到和鼠标点击区域相交点的位置、法线信息

    Grass Position

  2. 向Mesh写入草的数据，通过 AddGrassData 函数，传入草的位置和法线信息

    AddGrassData
编辑草的状态

步骤：

  1. 得到区域内的草
    Grass Check
  2. 修改对应的数据(顶点色、宽高)
    Grass Data Modify
删除草的状态

步骤：

  1. 得到区域内的草
  2. 从Mesh中删除对应数据
    HandleMouseDrag

区域可视化

HandleMouseDrag
